# 分享版说明

这个目录用于放置项目的静态分享页面，适合直接发给朋友预览，不需要对方运行 Flask 或 Python 后端。

## 生成分享数据

在项目根目录运行：

```bash
python scripts/build_share_bundle.py
```

运行后会自动生成：

- `share/data/share-data.js`：静态页面所需的预生成推荐数据
- `share/assets/images/*.jpg`：场景示例图副本
- `share/assets/diagrams/*`：产品设计图与技术架构图副本

## 如何分享

有两种最简单的方式：

1. 直接把整个 `share/` 文件夹打包发给朋友
2. 把 `share/` 上传到 GitHub Pages、Netlify 或 Vercel 这类静态托管

## 朋友如何打开

如果是本地文件夹，朋友可以直接双击打开：

```text
share/index.html
```

这个页面不依赖接口请求，离线也能查看。

## 页面内容

- 5 个场景的属性层推荐结果
- 每个场景的关键属性、流派先验、目标音乐特征
- Top-K 推荐歌曲列表
- 每个场景对应的示例图片
- 当前项目的产品需求设计图
- 当前项目的推荐技术架构图

## 限制说明

- 这是静态分享版，不包含实时上传图片推理
- 如果需要演示“上传图片后即时推荐”，仍然需要运行原来的 Flask 应用
